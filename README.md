# Dymn Baby

Android-додаток для швидкого відстеження режиму малюка у стилі Dymn Launcher.

## Що вже є

- головний екран із великим годинником і темною Dymn-палітрою;
- плитки підсумку за день: сон, активність, туалет, кількість записів;
- швидкі дії: почати/завершити сон, додати 30 хв активності, додати туалет із коментарем, додати нотатку;
- стрічка подій за сьогодні;
- локальне збереження записів у `SharedPreferences`.

## Збірка

```powershell
powershell -ExecutionPolicy Bypass -File .\build-debug.ps1
```

APK після збірки:

```text
app\build\outputs\apk\debug\app-debug.apk
```
